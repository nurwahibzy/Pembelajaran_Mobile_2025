2. Jelaskan maksud void async pada praktikum 1?

void async digunakan untuk membuat sebuah fungsi yang dapat menjalankan proses secara asinkron, yaitu proses yang dapat berlangsung di latar belakang tanpa menghentikan jalannya program utama. Dengan menambahkan kata kunci async, fungsi tersebut dapat menggunakan await untuk menunggu proses yang memerlukan waktu tertentu, seperti mengambil data dari internet atau menunggu delay, tanpa membuat aplikasi menjadi berhenti. Kata void menunjukkan bahwa fungsi tersebut tidak mengembalikan nilai apa pun, sehingga void async berarti fungsi yang tidak menghasilkan nilai, tetapi dapat menjalankan tugas-tugas secara asinkron.

3. Jelaskan fungsi dari anotasi @immutable dan @override ?

    * @immutable adalah anotasi yang berfungsi untuk menandai bahwa objek pada class tersebut tidak dapat diubah setelah dibuat, sehingga performanyalebih stabil.

    * @override adalah anotasi yang berfungsi untuk menulis ulang method atau properti dari kelas induk/superclass.
